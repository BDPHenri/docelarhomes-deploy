# Deer Valley Guide Mobile Design

**Date:** 2026-03-21

## Goal

Adapt the Deer Valley Basecamp guest guide for phone-first use without creating a separate route.

## Approved Direction

- Keep the existing guide URL responsive: `/guide/deervalleybasecamp/`
- Use a mobile-first dashboard at the top of the page for arrival essentials
- Show only four top actions in the mobile dashboard:
  - Wi-Fi
  - Directions
  - Check-in
  - Checkout
- Hide lower-priority first-screen items such as parking and support from the mobile dashboard
- After the dashboard, surface the two highest-value guest questions in separate blocks:
  - `Things to Do Highlights`
  - `Restaurant Highlights`

## Mobile Highlights

### Things to Do Highlights

- Deer Valley Resort
  - Copy should work year-round, not just winter
  - Approved copy direction: skiing, hiking, biking, and mountain days
- Best Trails Nearby
  - Cover Park City, Midway, and Heber together
- Jordanelle State Park

### Restaurant Highlights

- Back 40 Ranch House
- Midway Mercantile
- The Hub
- Each restaurant highlight should include a direct map link in-place

## Navigation

- Add a fixed mobile footer navigation with four items:
  - Home
  - Wi-Fi
  - To Do
  - Eat
- The mobile footer should help guests jump to the top dashboard and the two highlight sections quickly

## Constraints

- Keep the desktop experience intact
- Keep the visual family aligned with the Deer Valley landing page
- Publish only after the approved mobile direction is implemented and verified
